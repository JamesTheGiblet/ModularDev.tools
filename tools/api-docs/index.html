Key Features (Version 1.0)

The current version of Praximous is built on a powerful and stable foundation, designed to cater to both individual developers and large enterprise teams.

Unified Experience

We provide the right tool for the job, whether you're in a GUI or a terminal.

  • Web-Based Dashboard: A central, intuitive GUI to manage projects, configure repositories, trigger builds, and share documentation.
  • Powerful CLI: A full-featured command-line interface designed for developers and automation. Integrate documentation generation directly into your CI/CD pipelines.
  • On-Premises UI: The Enterprise version is a complete, self-hosted web application, providing maximum control and security.

Core Engine

The heart of Praximous is a robust and reliable documentation generation engine.

  • Multi-Language Rule-Based Extraction: We parse your source code and comments (JSDoc, Docstrings, Javadoc, Godoc) to build a precise, deterministic representation of your API.
  • OpenAPI 3.0 Specification Generation: We generate industry-standard OpenAPI 3.0 specs, unlocking the entire ecosystem of compatible tools.

AI Capabilities

We layer intelligent features on top of our core engine to enhance and accelerate your workflow.

  • Intelligent Description & Example Generation: Our AI assists in writing clear, human-friendly endpoint summaries, parameter descriptions, and request/response examples.
  • AI Guardrails and Validation: The AI acts as a reviewer, flagging incomplete descriptions and identifying mismatches to help you maintain high standards.
